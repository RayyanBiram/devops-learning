## Overview

The **Bash Battle Arena** is a command-line game designed by [CoderCo](https://coderco.io) to teach **Bash scripting** skills through fun, level-based challenges. Each level is structured as a "mission" that must be solved with a script, introducing progressively more advanced concepts as you advance.

> [!TIP]
> Attempt each level yourself before reviewing solutions.  
> Use `man`, `--help`, experimentation, and debugging flags like `set -x`.

---

## Skills Practised

- File and directory manipulation  
- Loops and conditional logic  
- Argument parsing  
- Log processing and monitoring  
- Backup strategies and retention policies  
- Writing defensive, production-style scripts  

---

## Key Lessons

- Automation requires predictable execution flow  
- Validation prevents silent failures  
- Small scripts can scale into powerful tools  
- Clean structure improves maintainability  
- Understanding shell behaviour matters more than memorising syntax  

---

## Reflection

Bash Battle Arena transformed scripting from isolated command usage into structured automation thinking. Each level reinforced execution flow, defensive scripting, and production-ready habits.
