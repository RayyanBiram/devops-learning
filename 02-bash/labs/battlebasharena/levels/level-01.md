# Level 1: The Basics

## Mission
Create a directory named `Arena` and then inside it, create three files: `warrior.txt`, `mage.txt`, and `archer.txt`. List the contents of the `Arena` directory.


## Script
```bash
mkdir arena
cd arena
touch warrior.txt mage.txt archer.txt
ls
```
